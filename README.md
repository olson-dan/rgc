# RGC

RGC is a trashy, thrown-together client for displaying [https://gemini.circumlunar.space/](Project Gemini) data.  I did it as an experiment with egui and async rust.

There are tons of bugs and unimplemented features.  Here is a list of gemini servers you can play with: [https://gemini.circumlunar.space/servers/](https://gemini.circumlunar.space/servers/)
